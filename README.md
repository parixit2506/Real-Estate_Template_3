# Luxury Real Estate Platform

A modern, high-performance real estate application built with the latest web technologies. This project features a premium user interface with smooth animations, interactive maps, and a responsive design.

## 🚀 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: 
  - [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
  - [Framer Motion](https://www.framer.com/motion/)
  - [Lenis](https://github.com/studio-freight/lenis) for smooth scrolling
- **Maps**: [Leaflet](https://leafletjs.com/) & [React Leaflet](https://react-leaflet.js.org/)
- **Routing**: [React Router v7](https://reactrouter.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🛠️ Installation & Setup

1.  **Clone the repository** (if applicable) or navigate to the project directory.

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Start the development server**:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.

## 📜 Available Scripts

- `npm run dev` - Starts the development server with HMR (Hot Module Replacement).
- `npm run build` - Builds the application for production to the `dist` folder.
- `npm run preview` - Locally previews the production build.
- `npm run lint` - Runs ESLint to check for code quality and errors.

## 📂 Project Structure

- `src/components`: Reusable UI components (Navbar, Footer, Cards, etc.)
- `src/sections`: Page sections and layout blocks
- `src/data`: Static data files
- `src/styles`: Global styles and configurations
- `public`: Static assets (images, icons)

## 🎨 Features

- **Responsive Design**: Optimized for all devices (Mobile, Tablet, Desktop).
- **Interactive Maps**: integrated property mapping.
- **Smooth Animations**: Premium feel with scroll and entry animations.
- **Modern Architecture**: Component-based structure using React 19.
